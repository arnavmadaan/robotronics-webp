### Robotronics: A Robotics Store and Price Tracker
Robotronics is a web-based platform for exploring and comparing robotics components based on project categories. Users can view recommended parts for different robot types, compare prices across multiple sellers, and track price trends using charts. The system uses JSON files as a mock backend and dynamically loads data using JavaScript. A simple cart system is implemented using localStorage for adding and managing selected items. This project demonstrates basic web development concepts with real-world application in robotics.
